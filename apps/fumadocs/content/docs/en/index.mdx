# Better Auth × MercadoPago

`better-auth-mp/plugin` is a [Better Auth](https://better-auth.com) plugin that adds full MercadoPago payment processing to your app with minimal configuration.

## What it includes

- **One-time payments** — create Checkout Pro preferences and track payment status
- **Subscriptions** — recurring billing via PreApproval (with or without plans)
- **Webhooks** — secure signature verification + automatic status sync
- **Database schema** — automatic migration of `mercadoPagoPayment`, `mercadoPagoSubscription`, and `mercadoPagoPlan` tables
- **Security** — built-in rate limiting, idempotency guards, and metadata sanitization

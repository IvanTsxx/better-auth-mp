# Better Auth × MercadoPago

`better-auth-mp` es un plugin de [Better Auth](https://better-auth.com) que agrega procesamiento de pagos con MercadoPago a tu aplicación con mínima configuración.

## Qué incluye

- **Pagos únicos** — crea preferencias de Checkout Pro y rastrea el estado del pago
- **Suscripciones** — facturación recurrente vía PreApproval (con o sin planes)
- **Webhooks** — verificación de firma segura + sincronización automática de estado
- **Schema de base de datos** — migración automática de las tablas `mercadoPagoPayment`, `mercadoPagoSubscription` y `mercadoPagoPlan`
- **Seguridad** — rate limiting, idempotencia y sanitización de metadata integrados
